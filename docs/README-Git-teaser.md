# Log Monitoring & Alert System

A lightweight, Python & Bash-based log monitoring tool designed for IT teams and small businesses.

Monitor application logs, system metrics (CPU, RAM, Disk, Network), and receive real-time alerts when thresholds are exceeded.

## Key Features
- Tracks INFO, WARNING, and ERROR entries in log files
- Sends email notifications for threshold breaches
- Monitors CPU, RAM, Disk, and network usage
- Generates daily reports for historical tracking
- Configurable via a single `config.conf` file
